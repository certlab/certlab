## Color Scheme

The navigation uses CSS variables for theming:
- `text-foreground`: Normal text
- `text-muted-foreground`: Secondary text
- `text-primary`: Active/hover states
- `bg-card`: Menu backgrounds
- `bg-accent`: Hover backgrounds
- `bg-primary/10`: Active backgrounds (10% opacity)
- `border-primary`: Active borders

## Icon Library

All icons are from Lucide React:
- Home, BookOpen, Award, Settings (menu triggers)
- Target, BarChart3, FileText, Database, Timer (menu items)
- Trophy, Medal, ShoppingCart, Folder, Wallet (menu items)
- Shield, Accessibility, Languages, Heart (special items)
