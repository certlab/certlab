# CertLab Keyboard Shortcuts

## Global Navigation

| Shortcut | Action | Context |
|----------|--------|---------|
| `Tab` | Move to next interactive element | Everywhere |
| `Shift + Tab` | Move to previous interactive element | Everywhere |
| `Enter` | Activate button or link | On buttons, links |
| `Space` | Activate button or toggle checkbox | On buttons, checkboxes |
| `Escape` | Close dialog or menu | When dialog/menu is open |

## Skip Navigation

| Shortcut | Action | Context |
|----------|--------|---------|
| `Tab` (from page load) | Focus "Skip to main content" link | On page load |
| `Enter` (when skip link focused) | Jump to main content | When skip link is visible |

## Tips for Efficient Keyboard Navigation

1. **Use Tab to explore**: Press Tab repeatedly to discover all interactive elements
2. **Look for skip links**: Many sites have "Skip to main content" links as the first Tab stop
3. **Enable full keyboard access** (Mac): System Preferences → Keyboard → Shortcuts → check "Use keyboard navigation to move focus between controls"
4. **Use browser shortcuts**: Learn your browser's shortcuts for common actions
5. **Enable caret browsing**: Press F7 in most browsers to enable navigating with arrow keys
6. **Screen reader quick keys**: If using a screen reader, learn quick navigation keys for headings, links, and form fields

## Accessibility Settings

### Windows

**Enable narrator**: `Windows + Ctrl + Enter`  
**High contrast**: `Left Alt + Left Shift + Print Screen`  
**Magnifier**: `Windows + Plus`

### macOS

**VoiceOver**: `Cmd + F5`  
**Zoom**: `Cmd + Option + 8`  
**Increase contrast**: System Preferences → Accessibility → Display → Increase contrast

### iOS

**VoiceOver**: Settings → Accessibility → VoiceOver  
**Zoom**: Settings → Accessibility → Zoom  
**Reduce motion**: Settings → Accessibility → Motion → Reduce Motion

### Android

**TalkBack**: Settings → Accessibility → TalkBack  
**Magnification**: Settings → Accessibility → Magnification  
**High contrast text**: Settings → Accessibility → High contrast text
