## User Flow

1. User logs in and lands on Dashboard
2. Study Timer section is prominently displayed at the top
3. User can:
   - Start a work session (default 25 minutes)
   - Switch session types before starting
   - Pause/Resume during a session
   - Reset the timer
   - View progress towards daily goal
   - Check quick settings
   - Review completed sessions
